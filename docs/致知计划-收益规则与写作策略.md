## 一、致知计划是什么（机制层面）

致知计划是知乎面向创作者的**优质原创内容现金激励**项目。核心逻辑可以拆成三步：

1. **报名 / 加入**：在创作中心开通，作品需声明原创、绑定到计划。
2. **内容被「计分」**：入选作品按**内容质量**和**互动表现**累计一个收益分（站内常表述为获得「盐粒 / 收益」）。
3. **结算成现金**：按周期（通常按月）把累计的收益换算成现金 / 可提现的奖励。

它和知乎其他变现方式的区别：

| 变现方式 | 钱从哪来 | 适合谁 |
|---|---|---|
| **致知计划** | 平台对**优质原创**的内容补贴（按质量+互动计分发现金） | 持续产出原创、垂直、有信息增量的创作者（**本项目的主战场**） |
| 盐选专栏 / 付费 | 读者付费阅读 | 有完整长篇 / 故事 / 课程的人 |
| 好物推荐 / 带货 | 商品佣金 | 测评、消费决策类 |
| 商单 / 品牌任务 | 广告主 | 有粉丝量的账号 |

> 对本项目来说，**致知计划是最对口的一条**：我们产出的是纯原创情感/成长类想法与文章，不带货、不接商单，正好吃「优质原创补贴」这条线。

## 二、收益是怎么算出来的（可优化的杠杆）

致知计划的收益不是只看阅读量，通常由两类因子相乘/加权决定，这两类都是我们能主动优化的杠杆：

### 杠杆 1：内容质量分（门槛 + 加权）
- **原创度**：必须是原创，搬运 / 洗稿 / 大量 AI 痕迹会被降权甚至取消资格。
- **垂直度**：账号长期聚焦一个领域（我们就是「自我觉醒 / 个人成长 / 情绪疗愈」），比东一榔头西一棒槌权重高。
- **信息增量 / 完读**：有观点、有具体场景、读得下去、读完率高的内容，质量分更高。
- **合规**：不违规、不擦边、不标题党。

### 杠杆 2：互动数据（流量分发后的真实反馈）
- 赞同、收藏、评论、转发、关注转化——其中**收藏**和**评论**对成长类内容尤其值钱，代表「读者认为值得反复看 / 被戳中想说话」。
- **早期互动速度**：发布后前几小时的互动，决定能不能进更大的流量池，进而决定曝光、进而决定总收益。

> 结论：**收益 ≈ 内容质量分 × 进入的流量池大小 × 真实互动密度**。
> 我们能控制的是「质量分」和「早期互动」，所以策略要围绕这两点设计。

## 三、对本项目可直接落地的写作策略

结合 `config.yaml` 的主题池（15 类成长/情绪痛点）和 `src/content_generator.py` 的写作约束，给出能提升致知收益的具体做法：

### A. 选题：从「主题」升级到「具体处境」
当前主题池是抽象标题（如「孤独不是问题，独处是能力」）。致知计划奖励**有信息增量**的内容，建议每条想法锚定一个**具体的、可被搜索/被代入的处境**：

- ❌ 泛：「如何与负面情绪共处」
- ✅ 具体：「周日晚上一想到周一就开始焦虑，我后来是这样过周日的」
- ✅ 具体：「微信几千个好友，深夜想找人说句话却划不动——这不是你的问题」

具体处境 = 更高完读率 + 更多「这说的就是我」的评论/收藏 = 两个杠杆同时拉高。

### B. 结构：前两行决定生死
流量池靠早期互动撬动，而早期互动靠**开头**。建议固定一个「钩子在前」的结构：
1. **第 1-2 行**：一个具体场景 / 一句戳中的话（不铺垫、不「其实」开头）。
2. **中段**：一个真实的转折或视角切换（不是道理，是「我后来发现……」）。
3. **结尾**：留一个**开放问题**或微小提醒——这是**主动钓评论**的位置，直接服务互动杠杆。

> 这一点和 `SYSTEM_PROMPT` 已有的「结尾留一个微小提醒或开放问题」是一致的，建议把它从「可选」变成「每条必做」，并优先用问句。

### C. 原创与「去 AI 味」是收益的生命线
致知计划对 AI 痕迹 / 搬运容忍度低。项目里已有的硬约束（禁鸡汤句式、禁对仗排比、禁名人名言、禁 emoji、禁读书笔记式元叙述）本质上就是在**规避被判定为低质/AI 内容**——这些约束要严格守住，它们直接关系到能不能拿到收益，不只是文风问题。

### D. 垂直度：别破圈，深耕一个标签
保持账号长期只发「自我觉醒 / 个人成长 / 情绪」一类内容，不要为了蹭热点发无关主题。垂直账号在致知计划里质量分更稳。`config.yaml` 的主题池已经天然垂直，保持即可。

### E. 发布时段对齐互动高峰
项目已设早 7:00 / 晚 18:00 两个 slot，正好卡通勤 + 下班的情绪内容消费高峰，有利于早期互动起量——这个设置对致知收益是正向的，保留。

### F.（建议新增能力）互动回收
致知收益吃互动密度，而项目 README「后续可加功能」里列的**互动监控（自动给评论点赞 / 简单回复）** 其实是直接的收益杠杆：评论区被作者回应会带来二次互动和关注转化。建议把它的优先级提到靠前。
