# 看过的视频，别再"看完就忘"——「视频助手」把每一帧知识都留下来

## 你是不是也这样？

深夜刷到一条讲得特别透彻的学习视频，干货密集、案例清晰，你一边点头一边想："这个一定要记下来"。可视频一关，第二天能想起来的只剩一句模糊的"讲得真不错"。

我们这一代人看过的视频，可能比读过的书还多——技术教程、行业分享、网课、播客、发布会……信息像流水一样冲过眼前，却很少在脑子里留下沉淀。问题不在于你不认真，而在于**视频是"线性"的、"易逝"的**：

- 想复习时，得拖进度条大海捞针，记不清那句关键的话在第几分钟；
- 想做笔记，又得反复暂停、手动敲字，一小时的视频抄下来累到放弃；
- 时间一长，那些真正有价值的内容，连同你当时的思考，一起被遗忘了。

看过 ≠ 学到。**没有被记录下来的知识，等于没有真正属于过你。**

## 「视频助手」帮你把视频变成可检索、可复习的文字财富

视频助手是一款 Android App，做的事情很简单也很关键：**把你手机里正在播放的视频，自动整理成一篇结构清晰的文章。**

它的工作流程是一条全自动的流水线：

```
录屏（采集系统声音）→ 抽取音频 → 语音转文字 → AI 总结成文
```

你只需要在看视频时点一下"录制"，剩下的交给它：

- **录屏采集**：默认只录系统播放的声音，不打开麦克风、不录你说话，干净纯粹；
- **语音转文字**：自动把视频里的讲解转写成文本，中文识别又快又准；
- **AI 总结成文**：用大模型把零散的口语转写，提炼成一篇有条理的文章，而不是一堆流水账。

最后你得到的，是一份**可以随时翻阅、随手搜索的文字稿**。三个月后想复习某个知识点，不必再从头看一遍视频——打开文章，关键词一搜就找到了。

## 为什么你会喜欢它

**看完不再是结束，而是沉淀的开始。** 每一条认真看过的视频，都会沉淀成一篇属于你的文章，攒下来就是你自己的知识库。

**复习快人一步。** 文字可以一眼扫过、可以全文检索，温习一小时视频的内容，可能只需要两分钟。

**留下你的"一点财富"。** 看完后随手在文章里补一句自己的想法、一个联想、一处质疑——这些当下的灵光，才是视频之外真正属于你的东西。日积月累，它们比视频本身更值钱。

**隐私让你安心。** 录制内容、音频和转写文本默认只存在手机的私有目录里；API Key 由你自己在设置中填写，仅保存在本机。只有当你主动开启云端转写时，相应数据才会上传到你选择的服务商，App 对此有明确提示。

**省心，且容错。** 流水线每一步的结果都会保存下来，某一步失败了可以单独重试、断点续跑，不会让你前功尽弃。
